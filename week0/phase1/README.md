phase1

This phase introduces the fundamentals of PySpark using simple DataFrame operations. It helps in understanding how data is created, filtered, and analyzed in a distributed environment.

---

Operations Covered

- Displaying all records
- Filtering data based on city
- Filtering data based on age
- Selecting specific columns
- Grouping data and counting records

---

Key Concepts

- SparkSession as the entry point
- DataFrame structure in PySpark
- Transformations like filtering and selecting
- Aggregations using groupBy
- Difference between transformations and actions
- Lazy evaluation

---

SQL Understanding

The same operations can also be understood using SQL concepts like SELECT, WHERE, and GROUP BY.

---

Learning Outcome

- Working with PySpark DataFrames
- Applying basic data transformations
- Understanding how big data tools process information
